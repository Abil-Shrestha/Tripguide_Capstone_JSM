## 	:toolbox: Getting Started

<!-- Installation -->
### :gear: Installation

Install with npm

```bash
  npm i
```
   

<!-- Run Locally -->
### :running: Run Locally

Clone the project

```bash
  git clone https://github.com/Abil-Shrestha/Tripguide_Capstone_JSM.git
```

Go to the project directory

```bash
  cd tripguide
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev 
```
